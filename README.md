# Simulador GNV 2025 (Vite + React + TS)

Cotizador rápido con la lógica **Actividad → Plazo → Modalidad** y tabla embebida.
Listo para desplegar en **Vercel** o **GitHub Pages**.

## Scripts
```bash
npm install
npm run dev
npm run build
npm run preview
```
Si publicas en GitHub Pages, edita `vite.config.ts` y define `base: "/SimuladorGNV2025/"`.
